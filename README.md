# Stock Exchange Matching Engine

## Features

- Price-Time Priority Matching
- Buy Order Book
- Sell Order Book
- Partial Fills
- Order Status Tracking
- Trade History
- Order History
- Lazy Cancellation
- Market Snapshot

## Tech Stack

- C++
- STL
- Priority Queue
- Unordered Map
- Unordered Set

## Build

g++ src/main.cpp src/Order.cpp src/OrderBook.cpp src/Trade.cpp -o exchange.exe

## Run

./exchange.exe
